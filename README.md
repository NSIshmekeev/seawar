# SeaWar

Десктопная версия морского боя на Kotlin. Поддерживает игру двух игроков по локальной сети (Wi-Fi).

## Состав

- `server` — WebSocket-сервер на Ktor, который сводит двух игроков в одну партию.
- `client` — десктоп-приложение (Compose Desktop) для подключения к серверу и игры.
- `shared` — общая модель игры, протокол сообщений и валидация флота.

---

## Игра по локальной сети (два компьютера в одной Wi-Fi)

### Шаг 1 — Хост запускает сервер

На компьютере **хоста** (одного из игроков):

```bash
./gradlew :server:run
```

При старте сервер выведет свой IP-адрес в сети:

```
==================================================
  SeaWar — Сервер запущен на порту 8080
==================================================
  Передайте второму игроку один из этих адресов:
  --> ws://192.168.1.5:8080/ws
==================================================
```

### Шаг 2 — Оба игрока запускают клиент

На каждом компьютере (в отдельном терминале):

```bash
./gradlew :client:run
```

### Шаг 3 — Подключение

- **Хост**: в поле «Адрес сервера» оставьте `ws://localhost:8080/ws`
- **Второй игрок**: замените `localhost` на IP-адрес хоста из вывода сервера, например `ws://192.168.1.5:8080/ws`

Оба нажимают **«Подключиться»** — сервер автоматически сведёт их в одну партию.

## Игра на одном компьютере

1. Запустите сервер:
   ```bash
   ./gradlew :server:run
   ```

2. В двух отдельных терминалах запустите два клиента:
   ```bash
   ./gradlew :client:run
   ```

3. Оба подключаются к `ws://localhost:8080/ws`.

## Расстановка флота

- Кликните по своему полю, чтобы поставить корабль нужного размера.
- Используйте кнопку поворота для смены ориентации (горизонтально / вертикально).
- Кнопка «Случайно» расставит флот автоматически.
- После расстановки нажмите «Подтвердить флот».
