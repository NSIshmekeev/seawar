# SeaWar

Десктопная версия морского боя на Kotlin.

## Состав

- `server` — websocket-сервер на Ktor, который сводит двух игроков в одну партию.
- `client` — Swing-десктоп приложение для подключения к серверу и игры.
- `shared` — общая модель игры, протокол сообщений и валидация флота.

## Запуск

1. Запустите сервер:

   ```bash
   ./gradlew :server:run
   ```

2. Запустите два клиента в разных окнах:

   ```bash
   ./gradlew :client:run
   ```

3. В клиенте укажите `ws://localhost:8080/ws`, нажмите подключение и дождитесь второго игрока.

## Расстановка флота

- Кликните по вашему полю, чтобы поставить корабль нужного размера.
- Используйте кнопку поворота для смены ориентации.
- Можно случайно расставить флот и подтвердить отправку.
